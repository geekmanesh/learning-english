# Learning English

My personal English learning repo a structured course track plus space for
free-form personal notes and study sources.

## Structure

```
src/       organized by level (a1 -> c2)
sources/      Material I'm studying from (books, shows, podcasts, courses)
personal/     Free-form notes, vocab log, mistakes journal
resources/    Reference material shared across levels (cheatsheets, links, tools)
```

### `src/`
The main folder. Each level (`a1`, `a2`, `b1`, `b2`, `c1`, `c2`) has its
own folder with a `README.md` describing objectives for that level, plus
skill files (`vocabulary.md`, `grammar.md`, `listening.md`, `speaking.md`) and
an `exercises/` folder. See [course/README.md](course/README.md) for the
overall roadmap and progress.

### `sources/`
Anything I'm learning from a book, a show, a podcast, an online
course. gets its own folder here with notes tied to that specific source.

### `personal/`
Not part of the structured course: day-to-day vocab I pick up, mistakes I
want to remember, journal entries written in English for practice.

### `resources/`
Shared reference material that isn't level-specific. grammar cheatsheets,
useful links, tools.
