# Course Roadmap

Structured progression through CEFR levels. Update the table as units are
completed.

| Level | Focus                              | Status      |
|-------|-------------------------------------|-------------|
| A1    | Basics, everyday phrases             | Not started |
| A2    | Simple exchanges, routine topics     | Not started |
| B1    | Independent use, familiar topics     | Not started |
| B2    | Complex text, fluent interaction     | Not started |
| C1    | Flexible, effective language use     | Not started |
| C2    | Near-native precision                | Not started |

Each level folder (`a1/`, `a2/`, ...) contains:

- `README.md` — objectives and what "done" looks like for this level
- `vocabulary.md` — word/phrase lists for this level
- `grammar.md` — grammar points and notes
- `listening.md` — listening practice notes/logs
- `speaking.md` — speaking practice notes/logs
- `exercises/` — practice exercises and drills
